# aliyundrive-checkin

## 说明

该脚本默认会每日执行 3 次，自动签到并领取奖励，请知悉

## secrets 配置

- `REFRESH_TOKENS`: 登录[阿里云盘](https://www.aliyundrive.com/drive)，在 `local storage` 中的 `token` 里查找 `refresh_token` 字段
  > 如果需要添加多个账号，请使用 `,` 分隔（英文逗号）
